Boris Final Effects Complete AE 6.0.2 Release Notes

Boris Final Effects Complete AE 6.0.2 supports the following host applications:

Macintosh:
Adobe CS3, CS4, CS5, CS5.5, CS6.
Boris RED 4.3x (via 32 bit FEC installer)

Windows:
Adobe CS3, CS4, CS5, CS5.5, CS6.
Boris RED 4.3x (via 32 bit FEC installer)

System Requirements:

Macintosh:
Max OSX 10.6 and higher

Windows:
Windows 7 32/64, Windows Vista 32/64, XP 32/64

What's New in FEC AE 6 (Summary of major changes since FEC 5):

• 3 New Transition Filters: Rectangular Scale Wipe, Super Dissolve, and RGB Dissolve.

• Re-engineered for 64 bit operating systems and hosts.

• Support for native After Effects lights.

• On-screen overlay widgets added to most filters for simpler heads up adjustment of key parameters.

• Addition of the Integrated Beat Reactor to many filters for audio driven keyframe generation.

• New Compare Mode feature added to many filters simpler comparison between filtered and unfiltered results.

FEC AE 6.0.2 – What's New :

• Support for AE CS6 and Premiere Pro CS6 hosts.

Known Issues and limitations:

The following is a listing of the known problems in the product at time of shipping:

  • The Beat Reactor function in FEC 6 is not available in the Boris RED 4.x host environment
  • FEC 6 On-screen Widgets are not available in Boris RED 4.x
  • Preset load/save and help buttons in FEC 6 are visible but non-operational in Boris RED 4.x
  • In Adobe Premiere Pro CS5, the Hair Thickness on-screen control is non-functional however the Hair Thickness parameter in the Effect Control Window operates as expected
  • In Adobe After Effects, lights that have been parented to a null object remain static when selected in FEC filters that support host lights
