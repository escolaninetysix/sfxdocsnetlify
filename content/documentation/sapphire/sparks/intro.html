Introduction to Sapphire Plug-ins for Autodesk systems

Sapphire Plug-ins is a package of image processing and synthesis effects for use with Autodesk's Flint, Flame, Inferno, Smoke, Fire, Flare and Burn systems.

It includes 171 "Spark" plug-ins with over 550 total effects. Each effect has many parameters which can be adjusted and animated for an unlimited range of results.

This document corresponds to version 2019.01 of 19-nov-2018. For updated information please check www.borisfx.com . Also see our Sparks support page for help with technical issues.
