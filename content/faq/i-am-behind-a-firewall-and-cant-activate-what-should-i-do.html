{ "date": "2014-07-01T18:50:00-04:00", "tags": [ "Mocha Pro", "Support FAQ" ], "title": "I am behind a firewall and can't activate my license, what should I do?" } If possible, turn off your firewall for a minute and activate online. Otherwise, you will need to send your Host ID and activation code into support for a manual license generation. If you have a floating license server, you will need to provide support with your HOST ID and SERVER NAME. Contact support now.
