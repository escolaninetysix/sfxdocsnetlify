  1. Add the project to the Render Queue:
  2. In the Render Queue twirl down the Output Module parameter and click on Lossless:
  3. In the Output Module window select Format Options:
  4. In Format Options select the codec that best suits your needs:
  5. Back in the Output Module check the box next to Resize, open the Resize to dropdown menu and select the size and frame rate that works for you:
  6. Hit okay, select your render destination, and start your render.
