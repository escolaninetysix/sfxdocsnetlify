No deactivation is necessary, simply uninstall Mocha from the machine. If you have run out of license activations, please contact support to request a license transfer. If you are working in a facility and wish to run MochaPro on more than one CPU, you might be interested in upgrading to a cross-platform, floating license. Please contact our sales teamfor more info.
