Optics-2022 : Filters : Sepia
 
Sepia
Description
Sepia
Creates a warm brown tone for that nostalgic feeling.
Category
Grads/Tints.
Controls
Presets
To select a preset, pick one from the Presets window.
Color
Amount
Determines the intensity of the color added to the image.
Opacity
Sets the opacity of the filter.
Preserve Highlights
Preserves the white areas of the image.
Exposure Compensation
Exposure Compensation adds back the brightness loss as a result of the filter application.
Grad
Sepia can optionally use a gradient that limits where the filter is applied. Grad is the transition area between the colored portion and the original image. Its direction, corners and size can be adjusted. Go to the Grad section of Common Filter Controls to see how the Grad controls work.
