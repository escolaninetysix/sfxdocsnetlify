Optics-2022 : Filters : Infrared
 
Infrared
Description
Infrared simulates infrared filters used in conjunction with infrared sensitive film or sensors to produce very interesting black and white images with glow in highlight areas.
Category
Stylize.
Controls
Presets
To select a preset, pick one from the Presets window.
Black and White
Selects the type of black and white filter to be applied to your color image. Go to the Black and White section of Common Filter Controls to see how the Black and White controls work.
Mist
Blend
Determines the blend mode to be used to create the glow effect.
Add
The glow is added to your image.
Screen
The glow is combined with the image using a Screen blend mode. This looks kind of like the Add blend mode, but highlights are retained.
Brightness
Sets the intensity of the glow.
Blur
Sets the softness of the glow.
Color Correct
Go to the Color Correct filter to see how the Color Correct controls work.
Matte
A matte is used to create the glow effect. Go to the Matte section of Common Filter Controls to see how the Matte controls work.
