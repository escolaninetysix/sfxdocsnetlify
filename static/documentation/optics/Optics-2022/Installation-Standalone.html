Optics-2022 : Installation
 
Installation
Standalone
1 Download Optics at www.borisfx.com/
2 Double-click on the file that was downloaded and run through the installation process.
3 Start Optics on Windows systems by selecting Programs > Boris FX Optics 2022 > Optics in the Windows Start menu.
or
4 Start Optics on Macintosh systems by going to /Applications/BorisFX/Optics 2022 and double-click on Optics.
The Optics user interface opens.
