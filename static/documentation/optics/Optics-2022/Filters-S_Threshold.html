Optics-2022 : Filters : S_Threshold
 
S_Threshold
Description
Sets the color channels of the image to full on or full off using a given softness and threshold. This can be used to increase the contrast of each color channel independently.
The S_Threshold filter comes from the Emmy award winning Boris FX Sapphire filter set.
Category
Color.
Controls
Presets
To select a preset, pick one from the Presets window.
Threshold
The source brightness value to use as the midpoint of the thresholding. This is often a middle gray around .5.
Threshold Add Color
Raises the thresholds on each color channel using this color. It has no effect when black.
Softness
The softness of the transition between full off and on. Increase for smoother transitions or decrease for sharper ones.
Soft Rel Red
The relative softness of the red thresholding.
Soft Rel Green
The relative softness of the green thresholding.
Soft Rel Blue
The relative softness of the blue thresholding.
Saturation
Scales the color saturation. Increase for more intense colors. Set to 0 for monochrome.
