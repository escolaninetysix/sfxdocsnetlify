Appendix D - Environment Variables
Environment variables can be used to customize certain aspects of Silhouette. See the Customization Guide for more information and a full list of environment variables and their descriptions.
Use the Environments preference to set environment variables directly inside Silhouette without having doing so in the operating system. Once applied, variables are immediately available in Silhouette, sub-processes, and scripts.
See the Environment Preference for more information.
