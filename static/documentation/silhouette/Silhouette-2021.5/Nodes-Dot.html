Silhouette-2021.5 : Nodes : Dot
 
Dot
Description
Displayed as a dot in the tree, the Dot node manually routes node connections and is useful for cleaning up complicated trees.
Create a Dot node by Alt-clicking on an existing node connection or add it from the Utility node group like any other node. To drag additional outputs from the Dot node, hold Alt over the dot and when the input/output ports appear, drag from the output port to the desired destination.
Node Group
Utility.
