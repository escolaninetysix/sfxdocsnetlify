Silhouette-2022 : Nodes : X-Ray
 
X-Ray
Description
Simulates the look of X-Ray images.
Node Group
Special Effects.
Controls
Presets
To select a preset, choose one from the Presets tab.
Black and White
Filter
The Filter pop-up selects the type of black and white filter to be applied to your color image. Go to the Black and White section of Common Node Controls to see how the Black and White controls work.
Brightness
Adjusts the brightness of the image. Positive values brighten, negative values darken.
Contrast
Adjusts the contrast of the image. Positive values increase contrast, negative values decrease contrast.
Gamma
Adjusts the gamma of the image. The gamma adjustment leaves the white and black points the same and only modifies the values in-between. Positive values lighten the midtones, negative values darken the midtones.
Color
Opacity
Sets the opacity of the color.
Color
The Color parameter sets the color of the x-ray through the use of a standard color picker and defaults to blue.
