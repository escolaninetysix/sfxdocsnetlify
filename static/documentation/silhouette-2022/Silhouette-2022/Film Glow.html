Silhouette-2022 : Nodes : Film Glow
 
Film Glow
Description
Generates a traditional film glow look when the frame is subjected to a strong light source causing the image to bloom.
Node Group
Light.
Controls
Presets
To select a preset, choose one from the Presets tab.
Glow Intensity
Sets the intensity of the glow.
Glow Color
Sets the color of the glow.
Glow Radius
Determines how much the glow spreads to surrounding regions.
Glow Radius X
The width of the glow.
Glow Radius Y
The height of the glow.
Glow Desaturation
Desaturates the glow.
Glow Threshold
Sets the pixel values that are affected by the glow. A value of 0 applies the glow to the entire image while a setting of 100 results in no glow.
Glow From Alpha
Generates the glow from the alpha channel instead of or in addition to the color channels.
Glow Boost Alpha
Controls how much the glow spreads outside of the original alpha channel.
Glow Under Alpha
Composites the image over the glow.
Edge Handling
Determines how the edges of the frame are handled.
Transparent
Allows the edge of the result to become transparent.
Repeat
Repeats edge pixels to generate opaque pixels along the edge of the result.
Reflect
Reflects pixels along the edge of the result.
RGB Independent Glows
Enables independent scaling of the glow for the Red, Green, and Blue channels.
Red Scale
Scales the red glow uniformly.
Red Scale X
Scales the red glow along the horizontal axis.
Red Scale Y
Scales the red glow along the vertical axis.
Green Scale
Scales the green glow uniformly.
Green Scale X
Scales the green glow along the horizontal axis.
Green Scale Y
Scales the green glow along the vertical axis.
Blue Scale
Scales the blue glow uniformly.
Blue Scale X
Scales the blue glow along the horizontal axis.
Blue Scale Y
Scales the blue glow along the vertical axis.
Glow Matte Type
Selects the channel from which the glow is generated.
Color
Uses the RGBA color channels and multiplies it by the source alpha channel.
Color Inverted
Uses the inverted color channel.
Luma
Uses the luminance channel.
Luma Inverted
Uses the inverted luminance channel.
Alpha
Uses the alpha channel.
Alpha Inverted
Uses the inverted alpha channel.
Extras
Threshold Color
The RGB values of this color are added to the threshold slider value to make an independent threshold for each channel. For example, if this is set to pure blue, the blue channel is not glowed, regardless of the channel values or the threshold slider setting.
Glow Apply Mode
Controls how the glow is combined with the image.
