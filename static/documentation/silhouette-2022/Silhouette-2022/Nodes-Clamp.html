Silhouette-2022 : Nodes : Clamp
 
Clamp
Description
Clamps out super-blacks and or super-whites by default.
Node Group
Color.
Controls
Min
Clamps out super-blacks.
Max
Clamps out super-whites.
Note: If Max < Min, then Max will internally be set to the maximum possible value. For instance, only super-whites would be clamped out.
