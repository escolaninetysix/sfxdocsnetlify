Roto Blend
The Roto Blend node smoothly blends the opacity of shapes together.
1 Load some source footage and create a session.
2 In the Trees window, add a Roto Blend node from the Nodes > Image group and connect it to the source node.
3 Single-click in the center of the Roto Blend node to view and edit it.
4 Create multiple shapes.

The shape's different opacities automatically blend from one to another.
