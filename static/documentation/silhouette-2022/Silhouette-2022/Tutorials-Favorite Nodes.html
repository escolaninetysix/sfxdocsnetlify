Favorite Nodes
Nodes can be tagged as a Favorite allowing them to be grouped separately in the Favorites tab of the Nodes window.
1 Select a node in the Nodes windows by single clicking on it.
2 Tag the selected node as a Favorite by pressing the Favorite icon at the top left of the Nodes window.
3 Click on the Favorites node tab to see the favorite nodes.
