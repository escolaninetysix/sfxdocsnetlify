Path Mask Point Editing
Shortcut
Action

Inserts a new point along the path
Delete key
Delete all selected points
Click and drag selected points
All selected points move
Click and drag an unselected point
Moves one point
Alt+click+drag on a point
Opens a slider to adjust the points tension. Left of center forces the path to curve through the point (Cardinal spline). The center position creates a corner point while the right position moves the curve towards the center of the path (B-Spline). If multiple points have been selected, they will all be set to the same tension.
Alt+click+drag+Shift on a point
Snaps the point tension to Cardinal, Corner or B-Spline positions
