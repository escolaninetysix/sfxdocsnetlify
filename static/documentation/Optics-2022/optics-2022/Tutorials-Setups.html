Setups
A Setup takes a snapshot of the filters and parameter settings applied to your image. Setups can be saved and loaded and are independent of the image they were originally applied to.
1 Select a filter, choose a preset and make some parameter adjustments.
2 Add additional filters if you like by clicking the Add Layer icon at the top of the Layers Window and then selecting another filter.
3 Select File > Save Setup or press Alt-S.
4 When the file browser opens, enter a name and click Save.
You can now apply this setup to a different image using File > Open Setup.
