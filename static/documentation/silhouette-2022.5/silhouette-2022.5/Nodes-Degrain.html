silhouette-2022.5 : Nodes : Degrain
 
Degrain
Description
Removes or extracts grain from images. If the grain is removed, edges can be protected. If the grain is extracted, it can be composited back into the image using the Grain Composite node.
Node Group
Filter.
Controls
Amount
Sets the amount of degraining.
Note: You may not see an accurate representation of the grain and noise removal in the viewer unless your timeline/composition is set to high quality and the viewer is set to a 1:1 pixel ratio.
Outputs
Output
Outputs the degrained image.
Grain
Outputs only the grain that is removed. Use this output in conjunction with the Grain Composite node.
