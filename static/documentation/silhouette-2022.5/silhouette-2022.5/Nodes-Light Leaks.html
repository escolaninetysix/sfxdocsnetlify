Silhouette-2022.5 : Nodes : Light Leaks
 
Light Leaks
Description
Light Leaks simulate the visual effect of light leaking through gaps in a camera and partially exposing the film/sensor.
Node Group
Light.
Controls
Presets
To select a preset, choose one from the Presets tab.
Geometry Group
Controls the overall geometric properties of the light leaks.
Position
Offsets the X/Y location of the light leaks.
Rotation
Rotates the light leaks.
Scale
Scales the light leaks.
Scale X
Scales the light leaks horizontally.
Scale Y
Scales the light leaks vertically.
Stylize Group
Various controls are provided to modify the light leaks.
Opacity
Sets the opacity of the light leaks.
Exposure
Changes the overall exposure of the light leaks.
Hue
Rotates the hue of the light leaks.
Saturation
Adjusts the saturation of the light leaks.
Blur
Blurs the light leaks.
Blur X
Blurs the light leaks horizontally.
Blur Y
Blurs the light leaks vertically.
Glow Intensity
Sets the intensity of the glow.
Glow Radius
Sets the size of the glow.
Glow Threshold
Determines which pixels contribute to the glow.
Flicker Amount
Adjusts the intensity of the flickering.
Flicker Speed
Adjusts the speed of the flicker.
Flicker Randomness
Sets the randomness of the flickering.
Texture Mutation Rate
Controls the rate at with the texture auto-animates over time.
Random Seed
Randomizes the light leaks.
Use Source Alpha
Controls whether or not the filter retains the original alpha channel of the clip. When enabled, the original alpha channel is preserved, whereas when disabled, the leak map will expand the alpha in regions where the leaks are present.
Apply Mode
Controls how the Ô¨Åltered image is composited with the source image.
Apply Mix
Controls the mix of the selected Apply Mode. If the Apply Mode is Normal, Apply Mix has no effect.
Generator Groups 1-4
Create up to four separately generated light leak maps, each with independent controls over all properties, which are then composited into a single combined light leak map before being applied to the source clip. The controls in each generator group are the same, but they may have different settings by default.
Enable Generator
Enables the generator.
Shape
Controls whether a shape mask is applied to the light leaks to limit the output to a smaller region.
None
No shape mask is applied.
Parabola
Limits the light leaks to an open parabolic shape.
Ellipse
Limits the light leaks to an ellipse.
View Shape
Displays the selected shape.
Shape Width
Sets the width of the shape.
Shape Height
Sets the height of the shape.
Fade
Controls the feather/fade of the shape boundary.
Position
Offsets the X/Y location of the light leaks and its shape mask, if used.
Rotation
Rotates the light leaks.
Scale
Scales the light leaks.
Scale X
Scales the light leaks horizontally.
Scale Y
Scales the light leaks vertically.
Threshold
Determines which regions of the light leaks are visible or transparent
Start Color
Sets the dark areas of the light leaks.
Mid Color
Sets the midtone areas of the light leaks.
End Color
Sets the bright areas of the light leaks.
Mid Color Position
Shifts the relative spacing of the light leak color gradient.
Blur
Adjust the softness of the light leaks.
Opacity
Sets the opacity of the light leaks.
Texture Amplitude
Increases the contrast of the light leaks which produces sharper, more extreme texture gradients and well defined distinctions between opaque and transparent regions.
Texture Offset
Adjusts how much of the light leaks fall within the threshold and the various color ranges.
Texture Frequency
Controls the rate at which the light leaks change with respect to image dimensions, with higher values yielding smaller light leaks.
Texture Aspect
Sets the relative stretch of the light leaks in the X/Y directions.
Texture Detail
Adjusts the level of detail in the texture pattern with large values generating more complex shapes.
