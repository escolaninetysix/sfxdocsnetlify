Time Blur
Description
Time Blur mixes frames together to create interesting motion effects. This node is also useful for smoothing out film grain and video noise which can cause problems when pulling a key or generating a matte.
Node Group
Time.
Controls
Frames
Sets the amount of frames to be blended.
Frame Window
Determines the positioning of the frame averaging window.
Past
The frame averaging window uses past frames.
Centered
Centers the frame averaging window around the current frame.
Future
The frame averaging window uses future frames.
