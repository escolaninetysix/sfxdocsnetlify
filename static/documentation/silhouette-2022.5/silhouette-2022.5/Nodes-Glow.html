Silhouette-2022.5 : Nodes : Glow
 
Glow
Description
The Glow filter creates glows around selected areas of the image.
Node Group
Light.
Controls
Presets
To select a preset, choose one from the Presets tab.
Glow
Blend
Determines the blend mode to be used to create the glow effect.
Add
The glow is added to your image.
Screen
The glow is combined with the image using a Screen blend mode. This looks kind of like the Add blend mode, but highlights are retained.
Brightness
Sets the intensity of the glow.
Blur
Sets the softness of the glow.
Color
The Color parameter sets the color of the glow through the use of a standard color picker. The default color is white.
Color Correct
Go to the Color Correct node to see how the Color Correct controls work.
Matte
A matte is used to create the glow effect. Go to the Matte section of Common Node Controls to see how the Matte controls work.
Alpha
Affect Alpha
The glow modifies the alpha channel.
Note: Affect Alpha requires that the image have an alpha channel.
Glow From Alpha
The glow is generated from the alpha channel instead of the RGB channels. In this case, the glow will not use color from the image and will typically be brighter. Values between 0 and 100 interpolate between using the RGB and the alpha.
Glow Under Source
Composites the image over the glow.
