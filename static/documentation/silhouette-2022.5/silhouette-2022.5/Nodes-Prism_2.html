silhouette-2022.5 : Nodes : Video Glitch
 
Video Glitch
Description
Video Glitch filter is an auto-animating filter for simulating the kinds of errors and artifacts that can be apparent during digital video playback.
Node Group
Special Effects.
Controls
Presets
To select a preset, choose one from the Presets tab.
Trigger Mode
Offers two modes for how and when glitches are triggered.
Automatic
The time at which glitches are triggered is based on the Glitch Interval and Interval Randomness parameters.
Intensity Threshold
A single glitch is triggered whenever the Glitch Intensity crosses a specified threshold (Trigger Threshold parameter) in a positive direction (from lower to higher value) by animating the Glitch Intensity.
Glitch Intensity
Sets the glitch intensity for all glitch groups.
Intensity Randomness
Randomizes the global glitch intensity.
Intensity Seed
Randomizes the glitch intensity.
Intensity Trigger Threshold
With Trigger Mode set to Intensity Threshold and as Glitch Intensity increases, this sets the threshold at which a glitch will be triggered.
Glitch Interval
Sets the glitch interval.
Interval Randomness
Randomizes the glitch Interval.
Interval Seed
Changes the randomized timing of all glitches.
Start at 0
Forces the first glitch to occur at the beginning of the effect even when the interval is randomized.
Glitch Duration
The length of the glitches.
Duration Randomness
Randomizes the glitch duration.
Duration Mode
Controls the type of duration.
Actual
The actual length of the glitches in time.
Average
The average length of the glitches in time.
Edge Handling
The Shift and Shake parameters can create instances where the image is offset and no image appears at the edge of the frame. Edge Handling offers options to Tile or Reflect the image to fill those areas.
Block Damage
With Block Damage enabled, a field of square blocks overlays the image during a glitch. In a typical application, some blocks will be more apparent than others depending on how they composite with the underlying image.
Enable Block Damage
Enables Block Damage.
Block Damage Intensity
Sets the Block Damage intensity.
Peak Position
Defines at the point within the duration of the glitch that displays the most blocks, with 0 being the beginning of the glitch, 100 the end, and 50 the midpoint.
Minimum Intensity
Sets the minimum amount of blocks.
Intensity Randomness
Adds more variability to the animation within the glitch duration--larger values produce a flickering appearance.
Block Run Length
Controls the number of blocks between gaps. A value of 0 generates completely independent blocks. A value of 100 generates one run of blocks with gaps on either side.
Block Size
Offers 5 options for the size of the blocks.
Block Saturation
Sets the block saturation.
Block Blend Mode
Blend options for compositing with the source image.
Pattern Amount
Sets the percentage of blocks displaying a pattern.
Pattern Complexity
Adds a variety of smaller patterns.
Pattern Opacity
Sets the maximum opacity of the patterns.
Vary Pattern Opacity
Randomly reduces the opacity of the patterns.
Vary Pattern Color
Tints a percentage of the patterns with colors.
Freeze Source
When enabled, the area of the source video within the block damage freezes and holds on the same frame for the duration of the glitch.
Block Seed
Randomizes the blocks.
Block Interval Seed
Changes the timing of the blocks.
Shift
With Shift enabled, horizontal bands of the image become horizontally displaced and skewed during a glitch.
Enable Shift
Enables shift distortions.
Shift Group Intensity
Sets the shift intensity.
Peak Position
Defines at the point within the duration of the glitch that displays the most shift, with 0 being the beginning of the glitch, 100 the end, and 50 the midpoint.
Minimum Intensity
Sets the minimum amount of shift.
Intensity Randomness
Adds variability to the animation within the glitch duration--larger values produce a flickering appearance.
Line Duplication
Artifacts where horizontal lines of pixels are duplicated and positioned adjacent to each other vertically creates the appearance of vertical stripes.
Shift Amount
Sets the peak horizontal shift and jitter amount.
Shift Density
Adjusts the overall density of shift bands.
Shift Run Length
Controls the number of shifted lines between gaps.
Skew
Defines the peak horizontal skew amount.
Jitter
Adds an additional randomized horizontal shift to each line.
Line Drop
Fades the opacity of every other line.
Line Drop Density
Randomly skips some line drops.
Intensity Seed
Randomization of shift intensity.
Shift Interval Seed
Changes the timing of the shift.
Shake Group
With Shake enabled, the image position is offset, skewed, and rotated over the duration of the glitch.
Enable Shake
Enables shake distortions.
Shake Group Intensity
Sets the shake intensity.
Peak Position
Defines at the point within the duration of the glitch that displays the most shake, with 0 being the beginning of the glitch, 100 the end, and 50 the midpoint.
Minimum Intensity
Sets the minimum amount of shake.
Intensity Randomness
Adds variability to the animation within the glitch duration--larger values produce a flickering appearance.
Shake Amount
Sets the amount of shake.
Shake X
Horizontal position of the shake.
Shake Y
Vertical position of the shake.
RGB Split
Controls how much the RGB channels are offset from each other in the direction of the shake movement.
Skew
The skew of the shake.
Rotate
The rotation of the shake.
Shake Intensity Seed
Randomization of shake intensity.
Shake Interval Seed
Changes the timing of the shake.
Flicker Group
With Flicker enabled, the brightness and/or saturation of the image flickers during glitches.
Enable Flicker
Enables flicker distortions.
Flicker Group Intensity
Sets the flicker intensity.
Peak Position
Defines at the point within the duration of the glitch that displays the most shake, with 0 being the beginning of the glitch, 100 the end, and 50 the midpoint.
Minimum Intensity
Sets the minimum amount of flicker.
Intensity Randomness
Adds variability to the animation within the glitch duration--larger values produce a flickering appearance.
Brightness
Controls the peak level of brightness.
Brightness Randomness
Sets the randomness of the brightness distortion.
Brightness Offset
Creates brighter or darker flickering. A value of 0 flickers brighter and darker, a value of 100 only flickers brighter, a value of -100 only flickers darker.
Flicker Saturation
Sets the flicker saturation.
Saturation Randomness
Controls the randomness of the saturation distortion.
Saturation Offset
Creates oversaturated or desaturated flickering. A value of 0 flickers both ways, a value of 100 only flickers oversaturated, a value of -100 only flickers desaturated.
Flicker Intensity Seed
Randomization of flicker intensity.
Flicker Interval Seed
Changes the timing of the flickering.
Use Background Color
Enabled so a color and not transparency will fill the areas created by shake, shift, and line drop.
Background Color
Sets the background color.
Curves Group
The parameters in the Curves group control a special timeline-like preview overlay that aids in visualizing the location and duration of the glitches over time. The various glitch preview curves are distinguished by color.
• Horizontal Lines: Red is Block Damage, Green is Shift, Blue is Shake, and White is Flicker.
• Vertical Blue Line: Indicates the current frame’s position within the graph.
Curve View
Displays the preview graph.
Off
The preview curve is disabled.
Render Over Black
The preview curve is composited over a solid black background.
Render Over Effect
The preview curve is composited over the glitched image.
Time View
Defines the time range of the preview graph.
