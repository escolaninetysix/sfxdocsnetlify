Silhouette-2022.5 : Nodes : DOD
 
DOD
Description
The DOD (Domain of Definition) node defines the size of the DOD. The DOD can be expanded or reduced. A smaller DOD will speed up all image processes. Setting the DOD larger than the Session size will pad smaller images and the ROI (Region of Interest) can access the additional image area. This is useful for creating oversize plates.
