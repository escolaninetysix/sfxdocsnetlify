Organizing The Tree With Dot Nodes
With complicated trees, the node connections can get a bit messy. To deal with this, the Dot node can manually route node connections in a cleaner fashion.
1 Create a Dot node by Alt-clicking on an existing node connection.
You can also add the Dot node from the Utility node group like any other node.
2 To drag additional outputs from the Dot node, hold Alt over the dot and when the input/output ports appear, drag from the output port to the desired destination.
Dot nodes strategically added to connections tidy up the tree.
