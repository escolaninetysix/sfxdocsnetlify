silhouette-2022.5 : Nodes : Turb Distort
 
Turb Distort
Description
Distorts the image by pulling it in a random manner.
Node Group
Warp.
Controls
Scale
Sets the size of the distortion.
Stretch
Stretches the distortion horizontally.
Angle
Rotates the distortion.
Amount
Sets the amount of distortion.
Speed
Automatically animates the distortion by multiplying it with Silhouette's frame number. So, if you set speed to 100, it uses the frame number as the time value directly. A speed of 200 makes it animate twice as fast, etc.
Time
Shifts the distortion animation by shifting it in time.
