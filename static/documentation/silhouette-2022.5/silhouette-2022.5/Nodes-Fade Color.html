Silhouette-2022.5 : Nodes : Fade Color
 
Fade Color
Description
Fades an image into a color.
Node Group
Image.
Controls
Amount
Sets the amount of the color.
Color
The color can be set through the use of a standard color picker.
Direction
Forward
The Amount transitions from the input image to the color.
Reverse
The Amount transitions from the color to the input image.
